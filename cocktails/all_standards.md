# All Solid Cocktails That I Didn't Come Up With

## Rum & Coke
- 1.5 oz Bacardi Rum
- 4oz Coke

#### Instructions
1. Fill tumbler with ice
2. Add Rum
3. Top off with Coke
4. Stir

## Whiskey Ginger
- 1.5 oz Whiskey (We used Jim Bean)
- 4oz Ginger Ale

### Instructions
1. Filll tumbler with ice
2. Add Whiskey
3. top off with Ginger Ale
4. Stir
